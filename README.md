# College

Site vitrine moderne pour une ecole, construit en HTML, CSS et JavaScript.

## Structure

- `index.html` : structure principale de la page.
- `public/assets/js/config.js` : informations reutilisables de l'ecole.
- `public/assets/js/main.js` : injection des donnees, galerie, animations, traduction FR/EN et interactions.
- `public/assets/css/app.css` : design personnalise du site.
- `public/assets/images/` : images utilisees par le site.
- `pages/inscription.html` : page de demande d'inscription secondaire.
- `pages/` : futures pages HTML.

## Modifier les informations de l'ecole

Ouvrez `public/assets/js/config.js`, puis changez les valeurs comme :

- `name`
- `tagline`
- `description`
- `phone`
- `email`
- `address`
- `heroImage`
- `gallery`
- `programs`

## Bibliotheques reliees

- Bootstrap
- Bootstrap Icons
- AOS
- Swiper
- Axios
- Particles.js
- GSAP
- Google Fonts : Plus Jakarta Sans et Fraunces

## Traduction

Le bouton `EN/FR` dans la navigation permet de changer la langue de la page d'accueil sans recharger le site.

## Utilisation

Ouvrez directement `index.html` dans un navigateur.
